CSS 기초

CSS란 무엇인가?

CSS는 Cascading Style Sheets의 약자로, 웹 페이지의 스타일을 지정하는 데 사용됩니다.

CSS의 주요 문법

CSS는 선택자, 속성, 값 등으로 구성되며, 다양한 스타일링을 적용할 수 있습니다.

CSS 사용 예제

간단한 CSS 예제 코드:


                /* 예제 코드 */
                body {
                    background-color: lightblue;
                }
